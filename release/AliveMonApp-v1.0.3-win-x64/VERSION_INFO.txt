Alive Mon App - Server Telnet Connection Monitor 
Version: 1.0.3 
 
Package Contents: 
- alive-mon-app.exe : Main executable 
- request/ : Sample CSV files for server list 
- results/ : Output directory for check results 
- run.bat : Launcher script (English) 
 
Documentation: 
- README.md : Project overview 
 
Key Features: 
1. Server Telnet Connection Check 
   - Network connectivity testing 
   - Port accessibility verification 
   - CSV file based batch processing 
 
2. Lightweight and Offline-Ready 
   - No external dependencies 
   - Works in completely isolated networks
