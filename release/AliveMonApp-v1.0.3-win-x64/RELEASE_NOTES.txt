======================================== 
  Alive Mon App 
  Server Telnet Connection Monitor 
  Release v1.0.3 
======================================== 
 
[Initial Release] 
Server Telnet Connection Check 
  - Check connectivity to remote servers 
  - Verify port accessibility 
  - CSV file based batch processing 
 
[Features] 
Lightweight Design 
  - No external dependencies 
  - Works in completely isolated networks 
  - Single executable file 
 
Multi-language Support 
  - English interface 
  - Korean interface 
 
[Installation] 
1. Extract the package to your desired location 
2. Add your server list CSV files to request/ 
3. Run run.bat (English) 
 
[CSV File Format] 
Required columns: 
- server_ip: Server IP address or domain 
- port: Port number 
 
Optional columns: 
- server_name: Server description
